100 Days of Code - Day 22

Day 22: June 4, 2020 #

Today's Progress #

Today I added updated the styling and navigation for my D3 chart gallery.

Thoughts #

new.css is a pleasant, minimalist, css framework. It's good at getting you going and making simple websites look decent.

I also added a basic breadcrumb nav to my histogram example which is a helpful touch.
